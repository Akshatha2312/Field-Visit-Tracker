# Field Visit Tracker

## Overview

Field Visit Tracker is a Django-based web application that helps organizations manage employee attendance and client visits efficiently.

## Features

- Employee Login
- Dashboard
- Attendance Management
- Client Visit Management
- Employee Profile Management
- Django Admin Panel
- Search Client Visits
- Mark Visits as Completed

## Tech Stack

- Python
- Django
- SQLite
- Bootstrap 5
- HTML
- CSS

## Installation

```bash
git clone https://github.com/Akshatha2312/Field-Visit-Tracker.git
cd Field-Visit-Tracker
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
